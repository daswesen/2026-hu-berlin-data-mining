# Data Mining – Konsultation IBI, Humboldt-Universität Berlin

Materialien für die Konsultationssitzung im Fernstudiengang Bibliotheks- und Informationswissenschaft (Sommersemester 2026).

## Inhalt

- `notebook.qmd` – R-Notebook mit sieben Data-Mining-Methoden am Book-Crossing-Datensatz (Assoziationsregeln, Collaborative Filtering, Clustering, Klassifikation, Record Linkage, Anomalieerkennung, Netzwerkanalyse)
- `Konsultation_IBI_Deck.qmd` – Quarto-Foliendeck (pptx) zur Sitzung

## Datensatz

[Book-Crossing Dataset](https://grouplens.org/datasets/book-crossing/) (Ziegler et al. 2005) — nicht im Repository enthalten, separat herunterladen.

## Voraussetzungen

R ≥ 4.3, Quarto, sowie die Pakete `tidyverse`, `arules`, `recommenderlab`, `rpart`, `rpart.plot`, `stringdist`, `igraph`, `visNetwork`.
